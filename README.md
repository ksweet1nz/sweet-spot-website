# Sweet Spot Advisory

Static website for [Sweet Spot Advisory](https://sweetspotadvisory.com.au) — boutique technology advisory for small and mid-sized B2B businesses.

## Structure

```
index.html          Home page
engagement.html     The engagement model
about.html          About Kevin Sweet
contact.html        Application form
assets/
  css/main.css      Design system and all styles
  js/main.js        Navigation, scroll reveal, form handling
  images/           Logo, mark, and favicon SVGs
```

## Stack

Plain HTML, CSS, and vanilla JavaScript. No frameworks, no build tools, no dependencies. Deploy by dropping the folder onto Netlify or any static host.

## Setup

1. Replace `FORMSPREE_ENDPOINT_HERE` in `contact.html` with your Formspree form URL
2. Drop the folder onto Netlify or serve with any static file server

## Design

- **Font:** DM Sans (Google Fonts) — weights 300, 400, 500, 700
- **Colours:** Navy #1A2B45, Amber #F59C29, Off-white #F1EFE8
- **Principles:** Flat design, no shadows, no gradients, max 2px border radius, typography-led hierarchy
